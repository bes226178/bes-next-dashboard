# 🍱 午餐選擇小工具

一個簡單有趣的午餐選擇工具，幫助你快速決定今天要吃什麼！

## 功能特色

### 🎲 隨機選擇
- 點擊「隨機選擇」按鈕，系統會從你的餐廳列表中隨機挑選一個
- 具有動畫效果，讓選擇過程更有趣

### 📝 餐廳列表管理
- **新增餐廳**：輸入餐廳名稱並點擊「新增」按鈕
- **刪除餐廳**：滑鼠移到餐廳卡片上，點擊垃圾桶圖示即可刪除
- **預設列表**：包含 12 個常見的午餐選項
- **重置功能**：可以一鍵恢復到預設餐廳列表

### 💾 資料持久化
- 使用 localStorage 自動儲存你的餐廳列表
- 下次開啟時會自動載入你上次的設定

### 🎨 美觀的 UI
- 漸層背景設計
- 響應式佈局，支援手機和電腦
- 動畫效果和過渡效果
- 使用 shadcn/ui 組件保持與專案風格一致

## 使用方式

1. 訪問 `/lunch` 頁面
2. 查看預設的餐廳列表，或新增你自己的選項
3. 點擊「隨機選擇」按鈕
4. 系統會隨機選出一個餐廳，並顯示結果

## 技術實現

- **框架**：Next.js 15 (App Router)
- **UI 組件**：shadcn/ui (Button, Card, Input, Badge)
- **圖示**：lucide-react
- **樣式**：Tailwind CSS
- **狀態管理**：React Hooks (useState, useEffect)
- **本地儲存**：localStorage API
